# [MTK]解决camera上下电顺序问题

## 笔记内容

**关闭了 Camera HAL 的异步关闭机制（ASYNC_CLOSING）**

关闭之后，camera power off 会变成**同步执行**，从而避免上下电顺序错乱的问题。

vendor下

```diff
--- a/mediatek/proprietary/hardware/mtkcam3/main/hal/device/3.x/device/Android.mk
+++ b/mediatek/proprietary/hardware/mtkcam3/main/hal/device/3.x/device/Android.mk
@@ -93,7 +93,7 @@ else
 endif

 ifneq (,$(filter $(strip $(MTKCAM_TARGET_BOARD_PLATFORM)), mt6855 mt6789 mt8189 mt8169))
-  LOCAL_CFLAGS += -DSUPPORT_ASYNC_CLOSING
+  += -DSUPPORT_ASYNC_CLOSING
 endif
```
