电池

healthd: battery l=80 v=4073 t=30.0 h=2 st=2 c=-294100 fc=7500000 cc=1 chg=a

可以这样拆：

- `l=80`：电量 80%
- `v=4073`：电池电压 4073 mV
- `t=30.0`：电池温度 30.0°C
- `h=2`：电池健康状态 2，也就是 `BATTERY_HEALTH_GOOD`
- `st=2`：电池状态 2，也就是 `BATTERY_STATUS_CHARGING`
- `c=-294100`：瞬时电流 -294100 µA，约 **294 mA**
- `fc=7500000`：满充容量 7,500,000 µAh，约 **7500 mAh**
- `cc=1`：循环次数 1
- `chg=a`：`a` 表示 AC charger 在线；源码里 `a/u/w/d` 分别对应 AC、USB、无线、dock。
